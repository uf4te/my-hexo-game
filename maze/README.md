# Features

- 移动端识别重力感应器进行移动，体验真实的重力感；
- 可自定义调整游戏难度、地图大小；
- 随机生成迷宫地图，每一次的都不尽相同；
- 桌面端也可通过方向键或 `W A S D` 进行移动，加速度固定；
- 地图较复杂时，可提示迷宫的通关出路；
- 无提示通关最大地图可触发游戏彩蛋🎉，尽情玩耍吧🎮；
